Library Management System (Java)

A simple console-based Library Management System built in Java.
It allows an Admin to manage books and a User to search or borrow books.

Features

Add, update, and remove books

Search by title, ISBN, author, or genre

Display all books or a single book

Borrow and return books

Separate Admin and User menus

Admin Login
Username: LocalAdmin
Password: Admin@9090

User Login
Username: LocalUser
Password: User@9090

How to Run

Compile:

javac TesterBookManager.java


Run:

java TesterBookManager

Description

This project uses simple Java concepts like classes, objects, lists, loops, switch-case menus, and custom exceptions to manage library operations.
